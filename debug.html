5. Console Debug

Open Developer Tools (F12) and check the console for debug messages. You should see:

  • "Shortly extension background script loaded"
  • "Tab updated with URL: [your-url]"
  • "Potential short link detected: [your-url]"
  • "Parsed prefix: [prefix] shortKey: [key]"
  • "Stored prefix: [prefix]"
  • "Available short links: [list of keys]"
